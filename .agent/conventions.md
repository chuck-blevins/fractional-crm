# Coding conventions (read on every task)
- Python 3.12, standard library only unless the task says otherwise.
- Type hints on all public functions/classes; one-line docstrings.
- Simplest correct implementation that passes the tests.
- NEVER edit, weaken, or delete anything under tests/. Tests are the spec.
- No unused imports — import only what the code references.
- Small, pure, well-named functions.
